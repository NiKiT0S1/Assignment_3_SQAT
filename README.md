Test Case: Automation of Select, Actions, and Waits on DEMOQA Platform

Использовались страницы “demoqa.com/select-menu” для теста Select Class и “demoqa.com/buttons” для теста Action Class

Implicit Wait реализован под видом ожидания загрузки chromedriver
Explicit Wait реализован под ожидание загрузки всех элементов на сайте
Fluent Wait реализован под ожидание сообщения о том, что элемент “Right Click Me” был успешно задействован – “You have done a right click” 
Action Class реализован под видом нажатия кнопки мыши/тачпада – один раз, double click, right click
Select Class реализован под выбор элемента из всплывающего Select меню

Шаги:
1) Открыть сайт https://demoqa.com/select-menu
2) Выбрать Select меню “Old select menu” со сменой цвета
3) Выбираем элемент под индексом 3 – это “Yellow”
4) В новой вкладке открыть ссылку https://demoqa.com/buttons
5) Дважды нажать по кнопке “Double Click Me”
6) Правой кнопкой мыши/тачпада нажать на кнопку “Right Click Me”
7) Один раз нажать на кнопку “Click Me”
